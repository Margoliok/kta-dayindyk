# КТА-ға дайындық — Магистратура

Магистратураға түсу емтиханына (КТА) дайындыққа арналған **офлайн оқу платформасы**.
Екі бейінді пән бойынша конспектілер, глоссарий және тест тапсырмалары.

## Пәндер
1. **Педагогика** — теориялық-әдіснамалық негіздер, тәрбие теориясы, дидактика, педагогикалық менеджмент, отбасы педагогикасы (5 тақырып).
2. **Қазақ тілін оқыту әдістемесі** — әдістеме негіздері, тарихы, қатысымдық әдіс, оқыту технологиялары, тіл білімі салаларын оқыту (13 тақырып).

## Мүмкіндіктер
- 18 тақырып бойынша қысқаша конспектілер мен глоссарий
- 300-ден астам тест сұрағы (Педагогика (2018) оқулығының тесттері ресми кілтпен тексерілген)
- Тақырып тесті (жауап бірден тексеріледі), пән бойынша емтихан
- **КТА сынама емтихан** — аралас сұрақтар, таймер, қорытынды балл және қателерді талдау
- Тақырыптар бойынша іздеу, прогресті сақтау (localStorage), ашық/қараңғы тема

## Ашу
`index.html` файлын браузерде ашыңыз — интернетсіз жұмыс істейді.

## Дереккөздер
Педагогика (2018) · Отбасы педагогикасы (2014) · Қазақ тілін оқыту әдістемесі (2016) · Әдістеме (2005) · Тілдік қатынас (2000, Ф.Ш. Оразбаева).
